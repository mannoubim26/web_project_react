# FitPulse

FitPulse is a mobile-friendly fitness web app built with React and Vite. It lets users track workouts, plan meals, monitor progress, and switch between multiple languages (English, Spanish, French, Italian).

## Getting Started

**1. Clone the repository**

```bash
git clone https://github.com/mannoubim26/web_project_react.git
cd web_project_react/my-fitness
```

**2. Install dependencies**

```bash
npm install
```

**3. Run the development server**

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
